# OKXB 策略设计 v1 — 锁定决策 + 待数据填充 (2026-06-08)

> 本文是与用户讨论后**锁定的策略蓝图**。它是计划,尚未全部实现;实现顺序见 §6,关键参数 H 由录制数据决定(§4)。

## 0. 一句话定义
**高频扫描、精选低频出手的「周期 H 方向概率预测」模型**:预测未来 H(分钟级)内的涨/跌**校准概率**,只在"概率够高 ∧ 预测幅度盖过成本 ∧ regime 合适 ∧ 可交易"时出手;**干净三选一出场**(止盈 / 止损 / 到 H 强平),不做中途 churn。

## 1. 已锁定决策(用户确认)
- **频率**:高频**扫描**(每0.5s全标的),**精选低频出手**,持仓 ≈ 分钟级。
- **信号性质**:预测"未来 H 内涨/跌的**校准概率 P**",**非**动量持续闩锁。"确定性高" = P ≥ 样本外验证过的阈值。
- **出场(三选一,干净)**:
  - **止盈** ≈ 预测幅度(且必须 > 往返成本);
  - **止损** = 论点失效(价格反向超出噪声) + 成本感知地板(≥2.5×成本);
  - **到 H 强平**(到达预测时间窗即兑现/离场,不恋战);
  - **去掉**:反转出场 / 衰减出场 / 中途信号抖动churn(与预测型冲突);移动止盈如保留,只用于"锁利",不作为churn出场。
- **标的**:**先加密单线跑通并验证**,股票永续后置(地区受限 + 不同博弈,单独建模)。

## 2. 盈利方程(一切决策的锚)
```
单笔期望 = p×W − (1−p)×L − 往返成本
往返成本 ≈ 10bps (2bps maker入 + 5bps taker出 + 滑点 + 半价差)
```
- 必要条件:**在 H 周期,预测幅度可靠地 > 成本**。
- 两个杠杆:**提高 edge**(H匹配的因子 + 重拟合 + regime择时)/ **降低成本**(尽量 maker 出场、提手续费档、减少次数)。
- **盈亏不对称比命中率更重要**:让盈利跑到止盈、快砍亏到止损 → 45% 命中也能盈利。

## 3. 目标架构(锁定哲学下的形态)
1. **因子(与 H 匹配)**:H=分钟 → 分钟动量、波动 regime、(股票)basis/funding/事件、订单流分钟累计;H=秒 → 微观结构(OBI/OFI/TI)。**先用数据定 H,再配因子。**
2. **打分 → 概率**:底层因子 → 拟合权重(Ridge/Logistic,正则化,简单模型)→ **校准成 P(涨@H)**(isotonic/Platt)。
3. **入场(全满足)**:P ≥ 阈值(OOS定) ∧ 期望净edge>0(预测幅度−成本) ∧ regime 合适 ∧ 可交易性达标 ∧ 非预热。
4. **出场**:§1 三选一。TP/SL 幅度由"预测幅度分布 + 波动 + 成本"定。
5. **仓位/杠杆(客观)**:张数 = 风险预算 ÷ 止损距离;杠杆 = 导出并钳 [3,50] ∩ 交易所上限;按"置信度超阈幅度 + 反波动"微调;逐仓。
6. **regime 过滤(可能最大的盈利杠杆)**:只在回测显示有 edge 的 regime 出手(波动带合适 / 价差正常 / 避开股票事件窗)。
7. **风控(已实现)**:单笔/日/总回撤 5%(−50 USDT)硬停;死手开关;数据老化 kill-switch。

## 4. 留给数据决定(录制 + 信号检验后填)
- **H ∈ {5s, 1min, 5min, 15min...}** ← 🔬信号检验「周期匹配诊断」:哪个周期 强信号幅度 > 成本。
- **哪些因子在 H 有 IC** ← Stage B-1 逐因子诊断(IC + 扣费后分桶净收益,多空/regime 分开)。
- **P 阈值、TP/SL 幅度** ← 样本外概率校准 + 预测幅度分布。

## 5. 当前代码需改造(待 H 确定后一起做,现在改早)
- 去掉 `executor.signal_exit` 的反转/衰减出场 → 纯 bracket(TP/SL)+ 到H强平。
- `edge_horizon_s` / `min_hold` / `max_hold` → 设为 H。
- 分数 → 校准概率(Stage B 产出新模型替换手设权重)。
- TP/SL → 由预测幅度 + 波动决定(替换当前 ATR/成本推导)。
- 入场去抖(persist/Schmitt latch)→ 改为"P 连续 N 拍超阈"的预测型去抖。

## 6. 实现顺序(= 验证"可实现"的路径)
1. **录够数据(含平静+活跃)** → 🔬信号检验 → 回答"有没有edge / 在哪个周期 / 够不够本"。
2. 若某周期够本 → **定 H** → **Stage B-1 逐因子诊断** → **Stage B-2 拟合权重 + 概率校准**。
3. 按 §5 改造出场/持仓/TP-SL 到 H;加 regime 过滤;尽量 maker 出场降成本。
4. **样本外 + DSR + PBO + 走步 + 成本×1.5/延迟+1s 压力** → 过 §7 闸门 → 小资金影子盘。
5. 加密跑通后 → 同框架复制到股票永续(单独模型,若账户可用)。

## 7. 上线闸门(不过不碰真钱;已实现于 calibrator/go_live_gate)
样本外 ≥300 笔、PF ≥1.25、胜率/平均RR 达标、DSR 过、PBO 低、成本×1.5 与延迟+1s 仍为正、最大回撤 < 预算 → 才小资金影子盘。

## 8. 否决条件(诚实止损,避免自欺)
若**所有周期 × 所有 regime,扣费+延迟后都无正 edge** → **不上线**。出路:换更强因子 / 只在高波动做 / 或承认当前成本下不可行(省下的亏损也是赢)。**最大风险是自欺**:所有结论必须样本外、扣费、可成交,而非样本内拟合。

## 9. 实证结论 — 分钟~小时级方向预测检验 (2026-06-08, 选项i)

用 **K线**(不是秒级tick)严格检验了"用合适分钟特征能否预测 5min~4h 方向、扣费后正"。
工具: `research/candle_data.py`(拉OKX历史K线+资金费) + `research/candle_research.py`
(H对齐特征/标签/IC/走步OOS多空回测) + `scripts/research_candles.py`。
**经 4-Agent 对抗审计**,修掉10个会"凭空造edge"的偏差(详见代码docstring与 §修复)。

**数据**: 25个加密永续 × 30天 × 5m K线 (面板21.6万行, 缺K线0%)。
**特征**: 多窗动量(15/30/60/120/240m)、波动regime、离均值、区间位置、量surge、vol归一动量、加速、RSI。
**判据**: 横截面多空『走步合并样本外』,组合层面 Newey-West t,资金费已扣,maker4/taker10bps。

| H | 毛(bps) | 净maker(bps) | NW-t | 判定 |
|---|---|---|---|---|
| 5min | +0.9 | −7.1 | −10.4 | 净负 |
| 15min| +1.4 | −6.6 | −3.3 | 净负 |
| 30min| +5.1 | −2.9 | −0.7 | 净负 |
| 1h | −4.5 | −12.5 | −1.8 | 净负 |
| 2h | −32.7 | −40.7 | −2.7 | 净负 |
| 4h | −26.6 | −34.6 | −1.0 | 净负 |

**结论(可信、保守)**: **这组标准分钟价量特征,在任何 H(5min~4h)、扣费后都无稳健可交易方向 edge。**
- 唯一真实效应 = **短周期横截面反转**(5–30min, xs_ic≈−0.034, IR−12, beta中性),但毛利仅 ~1–5bp « 成本 → 只有近零成本(maker返佣+大资金量)才可能收割,1000U散户taker不可行。
- **成本墙在长H并非瓶颈**(4h移动~200bp » 10bp成本);瓶颈是**预测力**:1h后组合毛利已转负,标准动量/反转在分钟~小时尺度对这批币基本无方向预测力。
- 可信度: 代码独立审计后,残留偏差(费率仅地板、未计半价差/滑点)只会让真实更差;走步OOS+聚类t;8币时4h那格的诱人+82bp在25币扩样后消失(−34bp),证明是小样本噪声、非edge。

**选择性曲线 + 逐H岭回归 (2026-06-08 追加, 回应"只做最自信top-X%能否转正")**: 加了 `fit_predict_oos`(逐H岭回归
特征→fwd_H=期望收益模型) + `selectivity_curve`(只做|pred|训练分布top-0.5/1/2/5/10/20%, OOS单边扣费, 组合NW-t)。
**结果: 模型测试IC≈0(各H +0.004~+0.009), 选择性各档随机变号, 唯一 t>2 的格(2h top20%)是~84格里的多重检验假阳
(与③横截面多空在2h显著为负矛盾)。加多重检验稳健门(模型ic>0.01 ∧ maker≥2档过 ∧ taker≥1档过)后判决仍❌。**
→ **"只做最自信top-X%"救不回**: 模型无预测力(IC≈0)时, 选"最自信"=选噪声。盈亏方程 E=pW−(1−p)L−cost 里没有 p-edge 可集中。
(过程中还抓到并修掉自己 verdict gate 的"任一格过即edge"漏洞=多重检验陷阱。)

**未否决的**: 这是【标准价量因子 × 这30天一个regime】的结论,**不**证明分钟~小时永不可做。
正交因子(**资金费/基差/跨所lead-lag/链上/期权/宏观**)未测。要继续做方向 → 必须先用数据验证这类正交因子,而非再调价量特征(已实证关闭)。

## 10. 实证结论 — 结构性 carry 验证 (2026-06-08, 选项B)

用 `research/carry_research.py` + `candle_data.py`(funding序列+现货K线) + `scripts/research_carry.py`
验证【资金费套利+perp-spot basis】(delta中性, 不撞预测墙)。**经 4-Agent 对抗审计**: 经济与现金流
被独立验证正确(空头收正费符号/无重复计/费用一次/篮子层面t), 修掉4个会【假阳性】的判据缺陷
(exit前视survivorship / n_reb门 / 现货流动性门 / NW-lag地板) + 改真实时间rebalance(兼容4h/8h混合)。
**跨所价差出范围**: Binance本机不可达 + 需第二账户跨境(安全约束禁VPN绕过)。

**数据**: 26个加密永续(perp+spot+funding齐全, 现货ADV≥$3M的24个) × 150天 × 1H。
**结果(top10正费币·delta中性·因果选币·扣两腿费)**:

| 指标 | 值 |
|---|---|
| 资金费持续性 IC(当期→下期) | **+0.44 (NW-t +29.6)** → 极可瞄准 |
| 最高资金费(150天均) | SUI +0.36bps/8h = **+3.9%/年** (主流币普遍2-4%/年) |
| 对冲残差(basis漂移) | +0.1~0.2bp → 对冲干净, 低风险 |
| 收到资金费 | 2天+1.7 / 5天+3.4 / 7天+5.0 bp (~0.7bp/天) |
| **净(7天maker)** | **−14.9bp = −7.8%/年** (NW-t −16, 0%胜) |
| 各持有×费档 | **全部净负** (NW-t −16~−76, 压倒性) |

**结论(可信)**: **资金费 carry 对 1000U OKX散户、扣两腿费后无可用 edge。** 资金费真实+干净+极持续(可瞄准),
但**比费墙薄**: OKX资金费小(市场有效, 主流币~2-4%/年), 而 cash-and-carry 需现货腿(费8-10bp是永续4-5倍),
两腿往返20-30bp; 要回本得持有~1个月, 届时≈0(还未计滑点/借币/挤压尾部)。
出路(均不适合现状): 高VIP降现货费 / 捕mania高费小币(但现货流动性差、容量小、尾部险) / 数月持有(放大风险)。

**三路径全部实证关闭(扣OKX散户费后)**: 秒级微观方向 / 分钟-小时标准价量方向 / 资金费carry。
框架按设计生效: 在模拟里、用真实数据、上线前抓出了亏损策略 —— 省下的亏损即收益。

## 11. 实证结论 — 正交因子离线验证 (2026-06-08, 影子盘计划第一步)

`research/orthogonal_research.py`(复用 candle_research 已审计原语)验证三个【与价量动量正交、数据可得、未测过】
的**方向**假设, 经**4-Agent 审计**(构造判定干净=无凭空造edge的前视; 修3个会假阳的缺陷:
判决门多重检验、funding面板改真实时间对齐、funding长持 n≥40 门; + embargo / lead排除领先资产自身 / basis std守护):

| 正交因子 | 结论 (扣费后, OOS, 组合NW-t, ≥2格门) |
|---|---|
| **基差做方向** (perp升水→回落) | 真实反转(xs_ic−0.08)但毛利~3.6bp « 成本 → 净 −4~−17bp, **0格过** |
| **大盘 lead-lag** (BTC/ETH→山寨) | xs_ic−0.04(且反向=过度反应), 净全负, 探索性不进门 |
| **资金费做方向** (拥挤→反转) | 加足样本+真实时间对齐后 **IC≈0, 净≈0/负, t≈0** → **0格过** |

**结论(可信)**: **三个正交方向因子扣费后均无稳健 edge。** 关键教训: 8币时 funding 一度显 +31bp(t1.0)→
powered 36币后 IC≈0、净负 = 典型小样本海市蜃楼; 旧 union 网格 bug 产生 −720/+1255bp 的荒谬幅度, 改真实时间对齐后消失。
**这正是"先离线证伪、再建系统"的价值**: 没有任何列出的正交因子值得现在去建影子盘引擎。
未测: 流动性冲击(需 L2 盘口深度, K线无); 不可得/不可执行(链上/期权skew/跨所)。

## 12. 实证结论 — 逐周期专用 ML 胜率模型 (2026-06-08, 用户要求 per-H 专用指标+ML)

`research/horizon_model.py`: 每个 H 用【按H缩放的专用指标库】(MA/BOLL/RSI/MACD/ATR/Stoch/量/K线形态/HTF, 19特征)
+ HistGradientBoosting 预测 P(涨@H) + 净化 walk-forward 样本外(embargo=H, 防重叠标签泄漏) + 选择性曲线。
**经 4-Agent 审计: 判定 TRUSTWORTHY(无前视泄漏, "no edge" 是真 null 非 bug; 测试集阈值经模拟验证不抬假阳率)。**

**结果(27币×30天, H=5/10/15/30/60)**: **模型确有真实但极弱的方向信号** —— AUC 稳定 **0.518~0.527**(>0.5),
高置信 top-2% 胜率 **53~58%**(真过50%)。**但扣费后样本外仍净负**: 5min 净 −6~−12bp(t显著负); 个别正格
(10/60min top2%)t<1=噪声; **无一格过门(双费 t>2, ≥2档一致)**。
**结论(可信)**: per-H 专用指标 + 非线性ML + 概率阈值选择, 也翻不过成本墙。正印证 `E=pW−(1−p)L−cost`:
能建 **55% 胜率** 模型, 但这些周期的移动幅度下, 55% 胜率 × 盈亏 < 成本 → 负期望。Phase 2(仓位/杠杆)不触发。

**系统化特征筛选 + 多模型比较 (2026-06-09, 用户质疑"是否变量筛选问题")**: `research/feature_lab.py`:
50-65 候选特征库 → 折内训练段筛选(覆盖率+共线性聚类保最强+RF重要性/Lasso/单变量IC排名聚合 top18)
→ 5 模型族(逻辑/ElasticNet/随机森林/HistGBM/MLP神经网)净化walk-forward比较。**经 4-Agent 审计判定 TRUSTWORTHY**
(筛选严格train-only无泄漏; 成本n_legs约定正确; 缺陷都在gate/多重检验层=只影响未来假阳, 不翻当前负)。
**结果(27币 H=10/15/30/60)**: **5 个模型族全部收敛 AUC≈0.52**(模型选择不是瓶颈), 自动筛出的最重要变量
正是用户建议的常规指标(Donchian区间/支撑承压res-sup/OBV/VWAP/BOLL)——**选对了变量仍翻不过成本**。10/15min扣费后≈0/负;
30/60min随机森林有 t≈2.2-2.4 的 maker-only 正格, 经审计判定=多重检验噪声(~50格预期~1假阳, taker即失效, 单格)。
**→ 变量筛选与模型选择都不是瓶颈, 成本墙是。**

**全谱实证关闭(扣 OKX 散户费, 全部经多Agent审计)**: 秒级方向 / 分钟价量方向(含选择性+岭回归) /
资金费carry / 三正交方向因子 / 逐周期专用ML胜率模型 / **系统化特征筛选×5模型族**。共性: 真实但极弱信号(AUC~0.52/IC~0.03), 被~成本墙吃掉。
绑定约束是【散户费率 vs 这些周期的移动幅度】这一结构, 不是"还没找到对的指标/模型"。唯一未穷尽的方程杠杆=出场/盈亏不对称(W/L)。
